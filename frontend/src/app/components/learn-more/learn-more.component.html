Here, I have listed a few things about this project:
Technologies Used
Angular 17, Express, RecordRTC Library, Firebase Realtime Database, Firebase Cloud Storage, and Prime NG
Features
User registration, login, and authentication. Protected routes, database, and cloud storage integration. Webcam video review, deletion, as well as play and stop functionality.
Problems Encountered
This was my first time using Angular. I encountered a problem using the RecordRTC library: "Uncaught TypeError: URL is not a constructor", when trying to use the library.
Solutions
I have experience with Django, flask, and react. Angular html templating reminds me of Django. I had to go into the node_modules and change some code to get that TypeError to go away.
